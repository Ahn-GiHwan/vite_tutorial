# vite_tutorial

새로운 번들러 vite를 알아보는 프로젝트

## 프로젝트 생성

npm create vite [project_name] -- template [project_template]
ex) npm create vite react -- --template react

<br/>

---

### **Template**

| JS      | TS         |
| ------- | ---------- |
| vanilla | vanilla-ts |
| vue     | vue-ts     |
| react   | react-ts   |
| preact  | preact-ts  |
| lit     | lit-ts     |
| svelte  | svelte-ts  |
